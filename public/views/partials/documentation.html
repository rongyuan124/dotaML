Device Sharing and Ownership

Every user of the device can share it with other users, or invite other people

Only device owner can delete other users

To help you in emergency situations, ts-support@tetrascience.com is always listed as a owner

Alert

Only device owner can set up global triggers that applies to all the users

You can choose email alert, text message alert or switch toggle alert
